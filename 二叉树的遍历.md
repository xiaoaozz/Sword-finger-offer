* 确定终止条件。

## 二叉树的中序遍历

```java
class Solution {
    public List<Integer> inorderTraversal(TreeNode root) {
        List<Integer> list = new ArrayList<>();
        dfs(root, list);
        return list;
    }
    // 1、确定递归函数的参数以及返回值
    public void dfs(TreeNode node,List<Integer> list) {
        // 2、确定终止条件
        if(node == null) {
            return;
        }
        // 3、确定单层递归的逻辑
        dfs(node.left, list); //左
        list.add(node.val);   //中
        dfs(node.right, list);//右
    }
}
```
